# Nomad App Store

Apple App Store를 React 기반의 SPA로 재구축한 앱입니다.

## Getting Started

이 지침은 개발 및 테스트 목적으로 로컬 컴퓨터에서 프로젝트 사본을 실행하는 데 도움을 줍니다.

### Prerequisites

```
node
yarn or npm
```

### Installing

1. Clone this repository

2. Install server dependencies
    ```bash
    $ cd nomad_store
    $ yarn add
    ```
    or
    ```bash
    $ cd nomad_store
    $ npm install
    ```

### Run the app

1. Start the server
    ```bash
    $ cd nomad_store
    $ yarn start
    ```
    or
    ```bash
    $ cd nomad_store
    $ npm start
    ```
2. Browse to `http://localhost:3000/`
